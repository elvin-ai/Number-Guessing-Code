# Number Guessing Game

## Author
Elvin Pillay

## Description 
This is a simple python code to guess a number within 5 attempts. This program generates a number randomly and then asks a user to guess it within five attempts. It also provides instant feedback to the user as a hint.This code is very ideal for programming beginners.

## Language Used 
Python

## Features 
1. Generates a random number
2. Beginner Friendly
3. Uses Python which is easy to learn and understand, especially for beginners
4. Instant feedback as a hint for the user
